# ott-news-web
ott-news-web

OTT News Web is a web application that brings you the latest updates, news, and trending content from popular OTT platforms. 

This project uses the **TMDB (The Movie Database) API** to fetch movie, TV show, and streaming data, making it easy to explore trending titles, top-rated content, and new releases.

### Features
- Browse trending movies and TV shows
- View details like ratings, genres, and release dates
- Search for content using TMDB API
- Clean and responsive web design

### Get Started
1. Sign up for a free account on [TMDB](https://www.themoviedb.org/) to get your API key.
2. Configure the API key in the project.
3. Run the web app locally and start exploring the latest OTT content.
